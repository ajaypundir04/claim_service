# Claim Service

This is a simple claim management service.

## Demo

Watch the demo video: [demo.mp4](https://www.dropbox.com/scl/fi/5j2m5m5w6so2bldi50af1/demo.mp4?rlkey=7xfaf15kuv3sqff526z14ixyt&e=1&st=cft99kyk&dl=0)

## Running the application

The application can be run using Docker Compose:

```bash
docker-compose up -d
```

This will start the application, which will be accessible at http://localhost:8080.

## API

The API documentation can be found at http://localhost:8080/swagger-ui.html.
